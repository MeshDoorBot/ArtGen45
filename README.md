# Mesh Art Generator

Next.js tool for generating Mesh DJ artwork in square post and Instagram Story formats.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Workflow

1. Select a show.
2. Choose square or story format.
3. Upload artwork.
4. Generate.
5. Use **Edit info** after the first render if details need correcting.
6. Download the finished PNG.
